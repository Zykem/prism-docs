# Adding replace/addon clothess

<div className='text-lg mt-2'
    style={{
        color: 'rgb(185, 198, 162)'
    }}
>If you use addon or replacement clothing, this feature automatically scans all your clothing items.</div>

<Callout type={'warning'}>We recommend performing these steps on an empty server with no players online, and avoiding any activity on your PC during the process.</Callout>

### Dependencies :
- [uz_AutoShot](https://github.com/uz-scripts/uz_AutoShot) (open source)
- yarn ( comes with fivem by default )

<Steps>

### Step 1

Download [uz_AutoShot](https://github.com/uz-scripts/uz_AutoShot) and place the resource in your resources folder.
Do not use a subfolder like `resources/[scripts]` as it will cause the script to malfunction.

### Step 2

Open your `server.cfg`, make sure `uz_AutoShot` and `illenium-appearance` are started, then restart your server and load in.

### Step 3

In-game, run `/shotmaker` to open the capture studio and start the batch capture for clothing, overlays, and tattoos. No copying required — the appearance UI pulls the images directly from `uz_AutoShot`.

### Step 4

Restart your server and load in!

</Steps>
